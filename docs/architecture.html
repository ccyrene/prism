Prism — Workload Identity Bus

Prism คือ เดมอนเล็ก ๆ ต่อโหนด (prismd) ที่เฝ้าดู Pod ของ Kubernetes แล้วแปลง label เป็น identity เลข 24-bit หนึ่งค่า เขียนลง eBPF map ที่แชร์ร่วมกัน (pinned) เพียงแผนที่เดียว เพื่อให้ระบบย่อย eBPF ทุกตัวอ่านได้แบบ O(1) — แผนที่นี้คือ “บัส” กลางของ identity.

Prism ไม่ใช่ ตัวจัดคิว (scheduler) และ ไม่ใช่ Cilium หรือตัวแทนของ Cilium; มันดึงเอา “แนวคิด identity” ออกมาเป็นชั้นกลางเป็นกลางที่ network, scheduling และ security ต่างก็มาเสียบใช้ร่วมกันได้.

Prism is a thin per-node daemon that turns pod labels into one 24-bit identity in a single shared, pinned eBPF map — read O(1) by many subsystems. It is not a scheduler and not Cilium.
